## ğŸ“˜ Project Overview
This project analyzes **loan default risk patterns** using **SQL and Power BI** to uncover financial and demographic factors contributing to loan non-repayment.  
It explores borrower characteristics such as **income level, employment type, education, credit score**, and **loan purpose**, revealing trends that can help lenders **predict and mitigate credit risk**.

Data was extracted and transformed using **SQL queries** for filtering, aggregation, and segmentation, then visualized through **Power BI dashboards**.  
The interactive report provides stakeholders with a holistic view of **loan performance, borrower profiles, and default behavior** over time.
